
A simple example of a restful web service


Dependencies
------------

  * Tested on Windows XP
  * Python 2.7

Product code requires the Python packages:

  * web.py 0.34

For development & running tests, also requires the Python packages:

  * nose 1.0.0
  * mock 0.7.0


Usage
-----

Start the server with the script::

    python restful-server.py

Open a browser and point at:

    http://localhost:8080
