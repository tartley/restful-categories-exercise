
A simple example of a restful web service


Dependencies
------------

  * Tested on Windows XP
  * Python 2.7
  * Python packages:
    * web.py


Usage
-----

Start the server with the script::

    python restful-server.py <options>

Check out the current state of the database with a browser pointed at:

    http://localhost:8080
